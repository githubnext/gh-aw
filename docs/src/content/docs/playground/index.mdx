## Getting Started Example

This is a simple workflow that creates a daily team status report. Follow along with the step-by-step guide to understand each component.

## Try It Yourself

Ready to experiment with GitHub Agentic Workflows? Here are some ways to get started:

1. **Clone the playground repository**: Fork or clone [gh-actions-playground](https://github.com/sombriks/gh-actions-playground) to experiment safely
2. **Install gh-aw**: Follow the [Quick Start guide](/gh-aw/setup/quick-start/) to install the extension
3. **Create your first workflow**: Copy the example above, save it as `.github/workflows/my-workflow.md`, and compile it with `gh aw compile`
