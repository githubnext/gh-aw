# Peli's Agent Factory - Article Series

This directory contains a comprehensive series of articles exploring the lessons, patterns, and practices learned from building and operating 145 autonomous agentic workflows in production.

## Articles

### [01. Meet the Agentic Workflows](01-meet-the-workflows.md)
A curated tour of the most interesting agents in the factory, organized by category (triage, quality, metrics, security, operations, etc.).

### [02. 12 Lessons from Peli's Agent Factory](02-twelve-lessons.md)
Key insights about what works, what doesn't, and how to design effective agent ecosystems. Includes challenges encountered and how to address them.

### [03. 12 Design Patterns from Peli's Agent Factory](03-design-patterns.md)
Fundamental behavioral patterns for successful agentic workflows, from Read-Only Analysts to Security & Moderation Agents.

### [04. 9 Patterns for Automated Agent Ops on GitHub](04-operational-patterns.md)
Strategic patterns for operating agents in the GitHub ecosystem: ChatOps, DailyOps, IssueOps, LabelOps, ProjectOps, ResearchPlanAssign, MultiRepoOps, SideRepoOps, and TrialOps.

### [05. Imports & Sharing: Peli's Secret Weapon](05-imports-and-sharing.md)
How modular, reusable components enabled scaling to 145 agents. Covers the import system, shared component library, and MCP server configurations.

### [06. Security Lessons from the Agent Factory](06-security-lessons.md)
Designing safe environments where agents can't accidentally cause harm. Covers core security principles, patterns, common mistakes, and incident response.

### [07. How Agentic Workflows Work](07-how-workflows-work.md)
The technical foundation: from natural language markdown to secure execution on GitHub Actions. Covers the three-stage lifecycle, compilation process, and runtime architecture.

### [08. Authoring New Workflows in Peli's Agent Factory](08-authoring-workflows.md)
A practical guide to creating effective agentic workflows, with examples and best practices from the factory. Includes authoring process, prompt writing, and common patterns.

### [09. Getting Started with Agentic Workflows](09-getting-started.md)
Begin your journey with agentic automation. Complete onboarding guide from installation through your first workflow and growth path.
